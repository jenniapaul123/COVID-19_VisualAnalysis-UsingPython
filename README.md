# COVID-19 Data Visualization (Altair)

## Overview
Interactive COVID-19 analysis using Python, Pandas, and Altair:
- Top 10 states by confirmed cases (bar chart)
- Confirmed vs Deaths (interactive scatter with dropdown)
- US county choropleth map by fatality rate (FIPS join)

## Tech Stack
Python, Pandas, Altair, vega_datasets, Jupyter Notebook

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Open the notebook:
   jupyter notebook Jennia_Paul.ipynb

## Dataset
Place `JHU.csv` in the project folder (or `data/` folder) before running.
